## Описание

Цель проекта — экспериментально проверить и проиллюстрировать три утверждения оригинальной статьи:

1. **Trade‑off между реконструкцией и распутыванием:** с ростом β качество реконструкции падает, а disentanglement сначала растёт, затем снижается (перевёрнутая U‑образная кривая).
2. **Влияние непрерывности данных:** снижение плотности выборки факторов ухудшает способность модели к распутыванию.
3. **Селективность латентных переменных:** β‑VAE формирует более специализированные латентные представления по сравнению с классическим VAE.

## Датасет

Все эксперименты используют **синтетический датасет 2D Shapes**: бинарные изображения 64×64 пикселя трёх геометрических фигур (квадрат, круг, треугольник) с варьируемыми независимыми факторами:
- позиция X (32 значения)
- позиция Y (32 значения)
- масштаб (6 значений)
- поворот (40 значений)

Датасет генерируется «на лету» при каждом запуске; загрузки внешних данных не требуется.

## Эксперименты

### Эксперимент 1: влияние β на распутывание и реконструкцию

**Идея:** обучить β‑VAE с разными значениями β и измерить точность распутывания (Disentanglement Accuracy) и ошибку реконструкции (MSE). Проводится два запуска: с малой моделью и расширенной архитектурой.

**Запуск:**
```bash
python experiment1_beta_tradeoff.py
python plot_experiment1.py
```

**Параметры:**
- β: 0.1, 0.5, 1, 2, 4, 8, 16, 32 (запуск 1) и β_norm: 0.5…64 (запуск 2)
- Эпохи: 200
- Размер датасета: 3000 и 10000 примеров на форму
- Latent dim: 10

**Метрики:**
- Disentanglement Accuracy (линейный классификатор)
- MSE реконструкции на валидационной выборке

### Эксперимент 2: влияние непрерывности данных

**Идея:** обучить β‑VAE с фиксированным β = 4 на версиях датасета с разной плотностью фактора вращения (40, 20, 10, 5 значений).

**Запуск:**
```bash
python experiment2_continuity.py
python plot_experiment2.py
```

**Параметры:**
- rot_step: 1, 2, 4, 8
- Эпохи: 200
- Размер датасета: 3000 примеров на форму

### Эксперимент 3: селективность латентов

**Идея:** обучить VAE (β=1) и β‑VAE (β=4), вычислить матрицы взаимной информации между 10 латентами и 5 факторами, сравнить степень специализации.

**Запуск:**
```bash
python experiment3_mutual_information.py
python plot_experiment3.py
```

**Метрики:**
- Disentanglement Accuracy
- Матрица Mutual Information (latent × factor)

## Результаты

**Эксперимент 1 (сравнение запусков):**

| β (запуск 1) | MSE    | Accuracy |
|--------------|--------|----------|
| 0.1          | 36.49  | 0.293    |
| 1.0          | 38.37  | 0.313    |
| 4.0          | 45.74  | 0.295    |
| 32.0         | 89.64  | 0.355    |

| β_norm (запуск 2) | MSE    | Accuracy |
|-------------------|--------|----------|
| 0.5               | 211.18 | 0.424    |
| 4                 | 627.34 | 0.296    |
| 64                | 627.05 | 0.316    |

**Основной вывод:** компромисс «реконструкция–распутывание» проявляется в обоих запусках; оптимальный β требует аккуратного подбора.

**Эксперимент 2 (непрерывность):**

| rot_step | rot_values | Accuracy |
|----------|------------|----------|
| 1        | 40         | 0.318    |
| 2        | 20         | 0.246    |
| 4        | 10         | 0.283    |
| 8        | 5          | 0.271    |

**Основной вывод:** лучшая точность при максимальной непрерывности, что согласуется с приложением A.8 статьи.

**Эксперимент 3 (селективность):**
- VAE (β=1): DisentAcc = 0.292; макс. MI = 0.89
- β‑VAE (β=4): DisentAcc = 0.296; макс. MI = 0.97

**Основной вывод:** повышение β усиливает специализацию латентных переменных даже при близкой общей точности.
